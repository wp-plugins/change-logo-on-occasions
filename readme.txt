If you want to change logos on certain occasions (e.g. Christmas, Halloween) just like Google does, this plugin is what you need!

== Description ==
Change Logo On Occasions adds the ability to set different images for various period of time. It adds \"Logos\" to WordPress administration menu, where you can manage logos (add, delete, schedule etc). Plugin is fully translatable, lightweight and very easy to use.

== Installation ==
1. Open Plugin installation page in WordPress administration
1. Search for \"Change Logo On Occasions\"
1. Install and activate the plugin
**OR**
1. Download the change-logo-on-occasions.zip file to your computer.
1. Unzip the file 
1. Upload `change-logo-on-occasions` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
Then place \"show_current_doodle_logo();\" in your template/theme.
